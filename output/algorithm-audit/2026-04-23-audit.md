## 5. Sinais que o Algoritmo Recompensa no Nicho

### Sinal 1 — Velocidade nas primeiras 24h (VPH de ignição)

- **O padrão:** Vídeos que obtêm VPH > 5 nas primeiras 24h recebem distribuição exponencial via Browse Features. No nicho de IA, os outliers têm VPH entre 5-340.
- **Evidência:** Os trending globais mostram VPH de 4.000-21.000 para canais grandes, mas mesmo canais de 83K subs (Mo Bitar) atingem VPH de 17.950 com content-market-fit forte.
- **Status no canal:** ❌ Não pratica — VPH mediano estimado em ~0,1. O canal não gera views suficientes na primeira hora para ativar distribuição.
- **Ação recomendada:** Criar sistema de notificação (comunidade, email, Shorts teaser) para gerar 50+ views na primeira hora. Postar no horário de pico da audiência mobile (19-21h BRT, que o canal já faz às 20h).

### Sinal 2 — Shorts como funil para Long-form

- **O padrão:** 91,9% do tráfego do canal vem de Shorts. O algoritmo de Shorts está distribuindo o conteúdo, mas a conversão Shorts → Long-form é mínima (12 views via Shorts Content Links = 0,27% de conversão).
- **Evidência:** 4.525 views de Shorts vs 173 views de inscritos (notificações) e 32 de suggested. O Shorts está gerando awareness mas não conversão.
- **Status no canal:** ⚠️ Parcial — Shorts funciona como topo de funil, mas o link para long-form não está otimizado.
- **Ação recomendada:** Em cada Short, incluir CTA visual + verbal para o vídeo long-form. Usar pinned comment com link. Testar end screen do Short apontando para o long-form.

### Sinal 3 — Retenção em Desktop como proxy de qualidade

- **O padrão:** Viewers em Desktop retêm 4x mais (102s vs 26s mobile). O algoritmo usa retenção como sinal primário de qualidade.
- **Evidência:** Desktop = 2,3% dos views mas 10,9% do watch time. TV = 7,3% dos views e 12,3% do watch time. Quando o viewer tem "lean-back intent" (TV/Desktop), o conteúdo performa muito melhor.
- **Status no canal:** ⚠️ Parcial — O conteúdo tem qualidade para reter em telas maiores, mas não está sendo descoberto nesses dispositivos.
- **Ação recomendada:** Otimizar para retenção nos primeiros 30s (hook mais forte) para melhorar a retenção mobile. Se a retenção mobile subir, o algoritmo distribuirá mais, incluindo TV/Desktop.

### Sinal 4 — Títulos com pattern-interrupt + curiosidade gap

- **O padrão:** Os vídeos do canal usam bons padrões de título: "A Neuralink Fez um Mudo FALAR com o Pensamento", "IA Reduziu Câncer em 50% e Ninguém te Contou", "IA vs Médicos: Quem Acha MAIS Câncer?". Estes seguem o padrão dos outliers do nicho: claim forte + curiosity gap.
- **Evidência:** Top outliers globais usam títulos opinativos/provocativos: "Microsoft accidentally told the truth about AI" (17.950 VPH), "How Dangerous Is Artificial Intelligence?" (5,12 VPH com 667K subs).
- **Status no canal:** ✅ Já pratica — Os títulos são fortes e seguem padrões comprovados. "IA Reduziu Câncer em 50% e Ninguém te Contou" é um excelente título com claim + exclusividade.
- **Ação recomendada:** Manter o padrão. Testar A/B com variações que incluam nomes próprios ou referências culturais ("ChatGPT", "Neuralink", "Google") para aumentar CTR.

### Sinal 5 — Conversão de viewer para inscrito

- **O padrão:** A taxa de conversão view → inscrito é de 0,18% (9 subs / 4.922 views não-inscritos). O benchmark para canais em crescimento é 1-3%.
- **Evidência:** 100% da audiência é não-inscrita — o algoritmo está servindo para audiência nova, mas quase ninguém converte. Se a conversão subir para 1%, seriam ~49 inscritos em 90 dias (8x mais).
- **Status no canal:** ❌ Não pratica — A taxa de conversão indica falta de CTA de inscrição ou que o viewer não vê motivo para se inscrever.
- **Ação recomendada:** Adicionar CTA de inscrição nos primeiros 60s de cada vídeo + no final. Criar "série" temática (ex: "IA & Medicina" como série) para dar motivo de voltar. Usar end screens com subscribe button.

### Sinal 6 — Frequência e consistência de publicação

- **O padrão:** O canal publica 1 vídeo long-form por semana (31/03, 07/04, 14/04, 21/04). Canais que crescem rapidamente no nicho mantêm cadência semanal ou bisemanal.
- **Evidência:** Breakout channels como SciTech World (+155% em 30d com 862 subs) e Mark Ai Guy (+168% com 5,9K subs) publicam com alta frequência. A consistência semanal é o mínimo para o algoritmo "aprender" o perfil de audiência do canal.
- **Status no canal:** ✅ Já pratica — Publicação semanal desde o lançamento. Manter.
- **Ação recomendada:** Manter cadência semanal para long-form. Considerar 2-3 Shorts por semana como complemento para alimentar o funil.

### Sinal 7 — Busca (Search) com alta intenção mas baixa retenção

- **O padrão:** YouTube Search traz 127 views (2,6%) mas apenas 20 min de watch time (0,16 min/view = ~9,4 segundos). Isso é uma retenção de busca extremamente baixa.
- **Evidência:** O viewer busca, encontra o vídeo, clica, mas sai em menos de 10 segundos. Isso envia sinal negativo ao algoritmo (bounce = "este resultado não satisfez a busca").
- **Status no canal:** ❌ Não pratica — Os vídeos aparecem em search mas não satisfazem a intenção de busca.
- **Ação recomendada:** Revisar quais termos de busca trazem tráfego (YouTube Studio > Traffic Source Details). Garantir que os primeiros 15s do vídeo validem a promessa do título/thumbnail. Se o tráfego de search é para Shorts que aparecem na busca, considerar que Shorts naturalmente têm baixo watch time em minutos.

## Resumo Executivo

### Diagnóstico de Distribuição: SHORTS-DEPENDENT

O canal **Marcus Maciel | IA & Ciência** está em fase de lançamento (4 vídeos, ~90 dias). O algoritmo está distribuindo os Shorts do canal (91,9% do tráfego), mas o conteúdo long-form **não está recebendo distribuição algorítmica** — zero Browse Features, mínimo Suggested. A audiência é quase 100% não-inscrita, o que é normal para canal novo.

### 5 Sinais Principais Identificados

1. **VPH de ignição praticamente zero** — O canal não tem massa crítica para ativar distribuição nas primeiras horas. Prioridade máxima: gerar views iniciais via Shorts teaser + comunidade.

2. **Shorts funciona, mas não converte** — 4.525 views de Shorts geraram apenas 12 cliques para long-form (0,27%). O funil está quebrado na conversão Shorts→Long.

3. **Retenção em desktop é 4x maior que mobile** — O conteúdo tem qualidade para reter, mas precisa capturar a atenção nos primeiros 15s para sobreviver ao consumo mobile.

4. **Taxa de conversão view→inscrito crítica (0,18%)** — Quase ninguém se inscreve. Sem CTA de inscrição efetivo, o canal não constrói base para gerar ignição futura.

5. **Search traz tráfego mas bounça** — 9,4 segundos de retenção no search. O conteúdo não está satisfazendo a intenção de busca ou os Shorts estão aparecendo em search com expectativa de vídeo longo.

### Oportunidades Imediatas

- **"IA na medicina"** como sub-nicho proprietário em PT-BR (o canal já produz isso)
- **"Machine learning"** como keyword de alto volume / baixa competição para search
- **Neuralink / interfaces cérebro-computador** como tópico trending com pouca cobertura PT-BR

### Próximo Passo Recomendado

Focar nas **hipóteses 1 e 2** (CTA de inscrição + Short teaser pré-lançamento) nos próximos 4 vídeos. Estas são as alavancas de maior impacto com menor esforço para sair do estado shorts-dependent.
