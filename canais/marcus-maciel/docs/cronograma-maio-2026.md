- [ ] Modelo de escrita consultado (`modelos-de-escrita/`)

- [ ] Títulos validados com `vidiq_keyword_research`
- [ ] Título final passa no checklist (<60 chars, <10 palavras, 1-2 CAPS, zero jargão, premissa)
- [ ] Roteiro aplica os 7 Princípios do DNA Narrativo
- [ ] Roteiro com 4 blocos escalados (transições invisíveis)
- [ ] Contra-argumento presente (se roteiro investigativo)
- [ ] Conclusão como crescendo emocional
- [ ] Duração 13-17 min (1.700-2.200 palavras)
- [ ] Thumbnail produzida (2-3 variações)
- [ ] Tags: 15-20 validadas por keyword research
- [ ] Legendas ativadas
- [ ] Post de Comunidade preparado

**Shorts (entregar até segunda para a semana):**
- [ ] Terça: Teaser ou Reprise pronto
- [ ] Quinta: Standalone ou Teaser pronto
- [ ] CTA específico em cada Short (nunca genérico)
