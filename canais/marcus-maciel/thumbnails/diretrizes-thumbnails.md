#### Configurações recomendadas no Nano Banana 2

* **Modelo:** Nano Banana Pro (Gemini 3 Pro Image) para máxima qualidade de texto; Nano Banana 2 (Gemini 3.1 Flash Image) para iterações rápidas.
* **Resolução:** 2K ou 4K (Nano Banana suporta 0.5K / 1K / 2K / 4K).
* **Aspect ratio:** **16:9** (obrigatório para thumbnail YouTube).
* **Reference images (opcional, até 14):** envie 1–2 fotos suas como referência de rosto se quiser aparecer na thumbnail, e indique no prompt: *"Use the uploaded photo as reference for the man's face and identity, keep facial features consistent."*

#### Dicas específicas de texto no Nano Banana 2

1. **Sempre entre aspas duplas** no prompt (`"ELE LEU SEU CÉREBRO"`). O modelo trata o conteúdo dentro das aspas como string literal a renderizar.
2. **Use CAIXA ALTA** no texto. Letras minúsculas com acento em português ainda têm taxa de erro mais alta; maiúsculas (Á, É, Í, Ó, Ú, Ã, Õ, Ç) renderizam melhor.
3. **Limite a 3–5 palavras.** Quanto mais curto, mais nítido.
4. **Repita o texto literal no fim do prompt** ("the only text visible is..."). Aumenta a fidelidade.
5. **Especifique a fonte por estilo, não por nome de arquivo.** "Impact / Bebas Neue style" funciona; "use the font Bebas Neue" não.
6. **Se o texto sair errado:** use a edição inline do Nano Banana ("change the headline to 'NOVO TEXTO', keep everything else identical") em vez de regerar do zero.

### Regras de Ouro para a sua "Outra IA":
*   **Regra do 1 Segundo:** A imagem deve ser compreensível instantaneamente. Se o prompt gerar muitos elementos, peça para a IA "simplify and focus on one object".
*   **Espaço para Texto:** Sempre defina **onde** fica o espaço negativo (`lower-left third`, `left half`, `upper-right corner`). "Negative space" sozinho é genérico demais.
*   **Expressão Humana:** Se você quiser aparecer, adicione: "A man with an expression of [EMOÇÃO: shock / intense curiosity / doubt] looking towards the main object, face occupying the right third of the frame."
*   **Texto na imagem (Nano Banana 2):** prefira **3 a 5 palavras em CAIXA ALTA**, sempre entre aspas duplas no prompt, com posição, tipografia, cor e tratamento (outline + drop shadow) explícitos. Defina **uma palavra-chave em vermelho** para hierarquia. Repita o texto literal no final do prompt ("the only text visible is..."). Se o texto sair errado, use a edição inline do Nano Banana ("change the headline to ... keep everything else identical") em vez de regerar do zero.
*   **Teste de mobile:** reduza a thumbnail para 320px de largura. Se não der para ler o texto e identificar o sujeito, refaça.
