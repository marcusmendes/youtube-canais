## Diagnóstico Geral

### Problemas recorrentes em TODOS os vídeos

| Problema | Impacto | Vídeos afetados |
|---|---|---|
| **Títulos tipo manchete** ("CONFIRMADO:", "DESCOBRIRAM:") | Anuncia resultado em vez de criar curiosidade. CTR baixo | V1, V4, V7 |
| **Títulos longos demais** (75-82 caracteres) | Truncados no mobile, sem impacto visual | V1, V4, V7 |
| **Hashtags dentro do título** (#câncer #curiosidades) | Desperdiça caracteres, visual amador | V5, V6 |
| **Sem CAPS cirúrgico** nos Shorts | Perde destaque visual no feed | V2, V3, V5 |
| **Tags insuficientes** nos Shorts (10-11) | Menor discoverability via busca | V2, V3, V6 |
| **Typo em hashtags** (#inteligênciaaritificialmedicina) | Hashtag quebrada, não agrega | V2, V3 |
| **Descrição curta demais** nos Shorts | Perde SEO via busca | V2, V3, V5, V6 |
