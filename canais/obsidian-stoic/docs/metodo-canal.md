## 5. Short Format Specification

### Duration

| Type | Duration | When to Use |
|---|---|---|
| **Standard** | 35-45 seconds | Default — ideal for list-style content |
| **Punchy** | 15-25 seconds | Single powerful insight or quote |
| **Deep** | 50-59 seconds | Story-driven or multi-step concepts |

### Visual Structure (Faceless)

```
[0s-3s]   HOOK — Bold text on screen + shocking statement (narration)
[3s-5s]   CONTEXT — Quick setup, "According to stoic philosophy..."
[5s-40s]  BODY — 3-7 points delivered with text + stock footage
[40s-45s] CLOSER — Provocative question or CTA
```

### Visual Elements

| Element | Specification |
|---|---|
| **Background footage** | Dark cinematic stock: marble statues, rainy cities, lone figures, shadows, Greek ruins |
| **Text style** | Bold condensed sans-serif, ALL CAPS key phrases, ember gold (#C8963E) or bone white (#F0EDE5) |
| **Text position** | Center screen, large enough to read on mobile |
| **Transitions** | Subtle zoom-ins, slow pans, fade cuts — never flashy |
| **Background music** | Dark ambient, low cinematic drones, tension-building piano |
| **Narration** | Deep male voice, calm but authoritative, measured pace |
| **Subtitles** | Always on — styled to match brand (white text, dark background strip) |

### Audio Layers

| Layer | Volume | Notes |
|---|---|---|
| Narration (voiceover) | 100% | Primary audio — clear, deep, authoritative |
| Background music | 15-20% | Dark ambient/cinematic, never overpower voice |
| Sound effects | 5-10% | Subtle whoosh on transitions, bass hit on key points (optional) |

## 6. Script Structure

### Template (35-45 second Short)

```
HOOK (0-3s):
"[Shocking statement or question that creates curiosity gap]"

SETUP (3-5s):
"[Brief context — tie to stoicism/psychology/power]"

BODY — Points (5-35s):
"Number one: [Point]. [Brief 1-sentence explanation]."
"Number two: [Point]. [Brief 1-sentence explanation]."
"Number three: [Point]. [Brief 1-sentence explanation]."
[...up to 5-7 points for longer Shorts]

CLOSER (35-45s):
"[Restate the core insight] + [Provocative question OR follow CTA]"
"Which one hit you the hardest? Comment below."
```

### Hook Templates (first 3 seconds — most critical)

**Category A — Forbidden Knowledge:**
- "Most people will never know this about [topic]..."
- "There's a dark psychology trick that [result]..."
- "Stoics discovered something 2,000 years ago that [modern relevance]..."

**Category B — Direct Challenge:**
- "If you do [common behavior], you're already losing."
- "Stop [common mistake]. Here's what powerful people do instead."
- "You're being manipulated right now and don't even know it."

**Category C — Listicle Tease:**
- "[N] signs someone is secretly [negative trait]."
- "[N] phrases that instantly [desired result]."
- "[N] stoic rules that will make you [transformation]."

### Closer Templates (drive engagement)

- "Which number was your wake-up call? Comment below."
- "Follow for more stoic truths most people can't handle."
- "Save this before it disappears from your feed."
- "Tag someone who needs to hear number [X]."
- "If this hit different, you already know what to do."

### Per-Short KPIs to Track

| Metric | Target | How to Measure |
|---|---|---|
| **Average % viewed** | >70% | YouTube Studio → Analytics |
| **First 3s retention** | >85% | Retention graph — the hook |
| **Swipe-away rate** | <30% | Inverse of retention |

| **Shares/Views ratio** | >0.3% | Viral signal |

### Decision Rules

| Signal | Action |
|---|---|
| Short gets >5x avg views in 24h | Create 2-3 follow-ups on same topic within 48h |
| Short gets <50% avg views after 48h | Analyze hook — likely failed in first 3 seconds |
| A pillar consistently underperforms | Reduce to 1x/week, redistribute to top performer |
| A pillar overperforms 3 weeks in a row | Increase to daily for that pillar, create sub-series |
| Engagement rate drops below 2% | Refresh hook style, test new title formula |

## 10. Production Pipeline

### Batch Production (recommended: produce 7 at a time)

```
Day 1 (Sunday): PLAN
├── Research trending topics via VidIQ (outliers + keyword research)
├── Write 7 scripts (one per day of the week)
├── Select title formula for each
└── Define visual direction per script

Day 2 (Monday): PRODUCE
├── Record all 7 voiceovers in one session
├── Source stock footage for all 7 (Pexels, Pixabay, Envato)
├── Edit all 7 Shorts (CapCut / Premiere / DaVinci)
└── Add subtitles, text overlays, music

Day 3-9: PUBLISH
├── Upload 1 Short per day at scheduled time (6-8 AM EST)
├── Add title, description, tags from templates
├── Assign to correct playlist
└── Monitor first 24h performance
```

### Tools Stack

| Tool | Purpose |
|---|---|
| **VidIQ** | Keyword research, outlier detection, competitor tracking |
| **CapCut / DaVinci Resolve** | Video editing (free) |
| **ElevenLabs / AI voice** | Voiceover (deep male voice, authoritative) |
| **Pexels / Pixabay** | Free stock footage (statues, cities, dark scenes) |
| **Canva** | Quick text overlays if needed |
| **YouTube Studio** | Publishing, analytics, A/B testing |
| **Notion / Sheets** | Script tracking, performance log |

## 13. Risk Mitigation

| Risk | Mitigation |
|---|---|
| Algorithm suppresses dark content | Frame as "educational awareness", add disclaimers when needed |
| Burnout from daily posting | Batch produce 7 Shorts in 2 days, schedule for the week |
| Content saturation in stoicism | Expand into adjacent niches: Machiavelli, Sun Tzu, behavioral economics |
| Copyright claims on stock footage | Use only royalty-free sources (Pexels, Pixabay, licensed Envato) |
| AI voice detected / penalized | Use high-quality AI voice with natural cadence, or consider real voiceover |
| Low initial traction | Commit to 90 days minimum before evaluating — algorithm needs data |
