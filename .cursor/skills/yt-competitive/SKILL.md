# YT Competitive — Análise Competitiva Avulsa

## Quando usar

O usuário invoca `/yt-competitive "tema"` (ex: `/yt-competitive "IA na medicina"`).

## Workflow

Lance o subagent **`/yt-competitive`** passando o tema fornecido.

O subagent analisa concorrentes via VidIQ (outliers, trending,
transcripts, comentários) e gera o briefing competitivo completo.

Após conclusão, apresente: vídeos analisados, erros/simplificações,
ângulos inexplorados, manifesto de diferenciação, insights da audiência.
